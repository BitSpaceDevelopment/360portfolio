04 — Logo System
Logo Usage
Dark Version — on dark backgrounds
Light Version — on light backgrounds
DO Use the dark logo on dark backgrounds only
DO Use the light logo on light backgrounds only
DO Maintain clear space equal to the cap-height of "B" on all sides
NO Never flatten the XR gradient to a single colour
NO Never place the dark logo on a light background or vice versa
NO Never stretch, rotate, skew, or add effects to the logo
NO Never rearrange the tagline "Immersive Technology Experts" — use it below or omit it
